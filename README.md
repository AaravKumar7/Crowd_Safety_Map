## Problem
Navigation apps optimize for speed, not safety.

## Solution
A crowd-verified safety map that highlights unsafe areas and provides AI-based emergency guidance.

## Features
- Crowd-verified unsafe zones
- Heatmap visualization
- AI emergency decision assistant
- Safe-route suggestions

## Tech Stack
React, Firebase Firestore, Mapbox

## Demo Flow
1. User reports unsafe location
2. Others verify the report
3. Area turns red on map
4. AI assistant gives guidance
